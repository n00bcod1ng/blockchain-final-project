# Blockchain Prescription System

A private, permissioned blockchain for securing medical prescriptions.
**Final Project for AdvTop Blockchain **

## Project Architecture
This project is split into two layers:
1. **Core Blockchain Engine:** Handles PoW consensus, SHA-256 hashing, and cryptographic signatures.
2. **Doctor/Pharmacy Portal:** A Flask-based web interface for issuing and verifying prescriptions.

## How to Run
1. Install Flask: `pip install flask`
2. Run the node: `python blochchain.py`
3. Run tests: `python tests.py`
